Ajax (Asynchronous javaScript and xml)是指一种创建交互式网页应用的网页开发技术

第一步:请求初始化,以搜索内容为例--相当于打开浏览器;
第二步:建立服务器,相当于输入网址;
    对象.open('参数1','参数2',参数3)
    参数1:method请求方式  get/post
    参数2:url 请求的地址
    参数3:是否异步  true表示异步,false表示同步
第三步:发送请求;
    对象.send();
第四步:做出响应.

readState:

  • 0:请求未初始化
  • 1:服务器已建立
  • 2:请求接受
  • 3:请求处理中
  • 4:请求已处理,且响应已就绪(这个时候说明我们可以拿响应数据了)

    status HTML状态码:

  • 200:代表成功
  • 404:找不到
  • 500:代表服务器出错了

javascript的代码

    
    <script type="text/javascript" src="js/jquery-1.12.4.js" ></script >
    <script type="text/javascript" src="js/jquery.easing.1.3.js" ></script >
    <script >
        $(function(){
            var $num=$('.banner-pic li').show().index();
            var $len=$('.banner-pic').children().length;
            var $timer;
            $('.banner-box').on('mouseleave',function(){
                $timer=setInterval(play,2000);
            })

            $('.banner-box').on('mouseenter',function(){
                clearInterval($timer);
            })
            function play(){
                $num++
                if($num===$len){
                    $num=0;
                }
                $('.banner-pic li').eq($num).fadeIn(1000).siblings().fadeOut(1000)
            }
        })
    </script >
